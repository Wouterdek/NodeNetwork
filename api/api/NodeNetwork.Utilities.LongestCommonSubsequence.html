GetChanges<T>(IList<T>, IList<T>)

Returns the changes to be made to oldList to reach the state of newList. First all items that are in oldList but not in the LCS of oldList and newList are removed. The list is then identical to the LCS of oldList and newList. Then all items that are in newList but not in the LCS of oldList and newList are added. The list is then identical to newList.
